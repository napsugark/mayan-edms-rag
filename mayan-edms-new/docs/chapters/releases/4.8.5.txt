Version 4.8.5
=============

Released: August 8, 2025


Summary
-------

Version 4.8.5 is a maintenance and refinement release that consolidates fixes and improvements from earlier branches, notably version 4.7.2. It delivers enhancements to the documentation system, user interface consistency, and build processes. This release also backports the fundraisers app and developer tooling updates.


Changes
-------


Documentation
^^^^^^^^^^^^^

Merged fixes and improvements from version 4.7.2 related to documentation handling and formatting.

Backported documentation file reference fixes for improved link integrity and maintainability.

Built the features section of the documentation dynamically from registered apps, ensuring accuracy and automatic updates as features change.

Fixed inter-document references to ensure cross-linked documentation is consistent and functional.

Updated .gitignore to exclude final built documentation artifacts, preventing accidental commits of generated files.

Removed an obsolete documentation chapter to reduce redundancy and outdated content.


Developer Tooling
^^^^^^^^^^^^^^^^^

Backported makefile changes to reference major release branches, ensuring that build and release automation aligns with branching policies.

Updated the version check code to use the current PyPI API, improving reliability when checking for new releases.

Backported the fundraisers app, making it available in the current branch for ongoing feature parity.


User Interface
^^^^^^^^^^^^^^

Persisted the success icon display after a file upload finishes, providing clearer feedback to users.

Removed hardcoded icons from the Dropzone file upload component, allowing for consistent theming and maintainability.

Unified the label for empty dropdown options, improving interface consistency across the application.
