Version 4.9.3
=============

Released: August 27, 2025

Summary
-------

Version 4.8.7 consolidates fixes and improvements from versions 4.8.5 and 4.8.6 while addressing compatibility issues with older installations, refining translation handling, and improving documentation clarity. This release ensures smoother upgrades, better internationalization coverage, and clearer presentation of important project information.

Changes
-------


Documentation
^^^^^^^^^^^^^

- Placed badges on the documentation index page to highlight important information at a glance.
- Fixed a broken link to the features chapter.
- Corrected the loading of the `config.env` file in the documentation.
- Applied minor layout and structural tweaks for clarity and consistency.


Dependencies
^^^^^^^^^^^^

- The coveralls version was downgraded from 4.0.1 to 3.3.1 due to compatibility issues in the package.
- The Coveralls library has been deprecated and will be removed in the next minor version.


Internationalization & Translations
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

- Updated the translation file detection process to ensure all JavaScript files are included in translation builds.
- Fixed string interpolation issues in the workflow app translations, ensuring placeholders render correctly in different languages.


Upgrade & Compatibility
^^^^^^^^^^^^^^^^^^^^^^^

- Added a **setting migration** to handle the removal of the old MIME type backend. This prevents errors on older installations and resolves GitLab issue #1187.
- Merged general changes and fixes from version 4.8.6 to keep branches aligned.
